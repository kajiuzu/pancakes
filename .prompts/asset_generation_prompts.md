# Asset Generation Prompts (2D Browser Game)

## Global Style Prompt
Bright, readable 2D browser game asset style, clean shapes, high contrast edges, warm friendly colors, minimal detail so elements stay clear on small mobile screens, no text labels inside icons, transparent background for sprites/icons.

## Background Prompt
Create a bright gameplay background for a 2D stacking game: soft sky gradient, subtle kitchen-inspired warm tones, low-noise decorative shapes only in far background, leave center area clear for gameplay readability, 16:9 composition.

## Player Prompt
Create a player sprite for a pancake catching game: simple white plate with blue rim and soft shadow, front-facing, crisp silhouette, high contrast outline, readable at 64px width, transparent background.

## Enemy Prompt
Create an enemy/obstacle sprite for a pancake game: mischievous burnt pancake disc with expressive face, dark brown body with orange edge highlight, thick black outline, readable at 64px, transparent background.

## UI Icon Prompt
Create a set of UI icons for small screens: score coin pancake icon, heart/life icon, warning icon, mission star icon. Flat bright style, bold outlines, 64x64 each, transparent background.
